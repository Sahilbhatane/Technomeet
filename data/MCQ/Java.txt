Java MCQ
Easy level :
1. Which method is the starting point of a Java program?
A. start()
B. main()
C. run()
D. init()
? Answer: B

2. Which keyword is used to inherit a class?
A. implements
B. inherit
C. extends
D. super
? Answer: C

3. Which of these is a checked exception?
A. ArithmeticException
B. NullPointerException
C. IOException
D. ArrayIndexOutOfBoundsException
? Answer: C

4. Which of the following cannot be declared static?
A. Variable
B. Method
C. Constructor
D. Block
? Answer: C

5. Which collection does NOT allow duplicate elements?
A. ArrayList
B. Vector
C. HashSet
D. LinkedList
? Answer: C

Medium Level :

6. Which keyword prevents method overriding?
A. static
B. private
C. final
D. protected
? Answer: C

7. Which of the following is immutable?
A. ArrayList
B. String
C. StringBuilder
D. StringBuffer
? Answer: B

8. Which access modifier allows access only within the same package?
A. private
B. protected
C. default
D. public
? Answer: C

9. What will be the output? (Logic)
int a = 5;
System.out.println(a++ + ++a);
A. 10
B. 11
C. 12
D. 13
? Answer: C
(5 + 7 = 12)

10. Which memory area stores objects in Java?
A. Stack
B. Heap
C. Method area
D. CPU register
? Answer: B

11. What will be the output? (Logic)

    System.out.print(i++);
